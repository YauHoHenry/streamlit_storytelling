# Re-interpret Story of Snow White: The Unsleeping Echo

**Author:** Ng Chau Yi, Sharon

## Introduction

This project is an interactive Streamlit retelling of Snow White in a dark fairytale style.
The story follows Snow White through a psychological mirror-world where beauty, hunger,
self-image, and perception begin to blur. Readers move through a linear chapter journey
from a cover page to a final ending scene, with cinematic visuals and atmospheric styling
designed to support the emotional tone of each chapter.

## Setup

### 1) (Recommended) Create and activate a virtual environment

```bash
python -m venv .venv
source .venv/bin/activate
```

### 2) Install dependencies

```bash
pip install -r requirements.txt
```

### 3) Run the app

```bash
streamlit run app.py
```

Open the local URL shown in terminal (usually `http://localhost:8501`).
